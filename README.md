# 셈여림 잠입 작전: 보안문을 열어라!

초등학교 4학년 음악 수업용 셈여림 웹앱입니다.

학생은 비밀 요원이 되어 `p`, `mp`, `mf`, `f` 셈여림 암호를 소리 크기로 맞추고 보안문을 통과합니다. 모든 문을 통과하면 셈여림 보물을 얻습니다.

## 배포 링크

GitHub Pages 배포가 완료되면 아래 주소에서 열 수 있습니다.

https://youu5736.github.io/-/

## 주요 기능

- 마이크 입력으로 `p`, `mp`, `mf`, `f` 셈여림 감지
- 동요 반주 모드: 나비야, 학교종, 작은별
- 보안문 잠입 미션과 작전 진행도
- 최종 보물 금고 엔딩
- 모든 기능은 브라우저 안에서만 실행되며 소리는 저장되거나 전송되지 않습니다.

## 참고

마이크 기능은 HTTPS 환경에서 가장 안정적으로 작동합니다. GitHub Pages 주소는 HTTPS라서 로컬 파일보다 마이크 권한이 잘 열릴 가능성이 높습니다.
